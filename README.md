<h2>📌 Usage</h2>
<p>Run the tool and select any option from the menu:</p>

<pre>
[ ? ] Choose A Option

[ 1 ] IP TRACK       → Enter any IP address to get details
[ 2 ] MY IP TRACK    → Track your own public IP
[ 3 ] EXIT           → Quit the tool
</pre>
